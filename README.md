# Area & Unit Conversion Calculator

This is my first desktop GUI application built using Python and Tkinter! It is a two-in-one app that lets you switch between calculating the area of geometric shapes and converting different types of units.

---

## ⭐ Features

- **2-in-1 Interface:** Easily flip between the "Area Calculator" tab and the "Unit Converter" tab.
- **Smart Fields:** The input boxes automatically change based on the shape or unit type you select (e.g., selecting a Triangle shows "Base" and "Height").
- **Dark Theme:** Custom dark-mode design with a unified widget look.

## 📂 Project Files

I organized the project folders to separate the user interface from the math logic:

- `main.py` - The main file you run to open the app window and layouts.
- `src/area.py` - Contains all the math formulas for calculating shape areas.
- `src/conversion.py` - Contains the logic and math for converting units.
- `assets/logo.png` - The custom icon image used for the window corner.
